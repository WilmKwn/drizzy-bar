# Drake Bar Ranker

Vote on the best Drake lyrics and see what other people think too

## Project Screen Shot(s)

## Tech Stack
### Frontend
* React.js
* Tailwind CSS

### Backend
* Golang
* PostgreSQL
* Docker

## Installation and Setup Instructions

Clone down this repository. You will need `Golang` and `Docker` installed on your machine.  

# Frontend
Installation:
`npm install`  

To Start Server:
`npm start`  

To Visit App:
`localhost:3000/`
